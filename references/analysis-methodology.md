## Section 0：封面仪表盘

> 30秒扫读版，让人在电梯里完成对今日市场的基本判断

**必含要素**：4大指数（沪指/深成/创业板/科创50）、情绪评分、今日定性一句话、3个关键词、明日核心情景概括

**"今日定性"写作标准**：
- ❌ "市场整体震荡，科技板块表现较好" → 平铺直叙，无洞察
- ✅ "情绪脉冲后的理性回归——开盘杀跌是虚晃，午后科技反攻才是主逻辑"
- ✅ "存量资金的防御性切换，上涨缺乏增量支撑，反弹可期但高度有限"

## Section 1：宏观定价扫描

**核心升级**：引入"定价完成度"概念

### 有效事件过滤矩阵

| 事件 | 传导路径 | 受影响资产 | 盘面印证 | 定价完成度 |
|------|---------|-----------|---------|-----------|
| 事件A | A→B→C | 板块/个股 | 涨跌数据 | 已充分/未充分/过度 |

**定价完成度判断**：
- **已充分**：板块已大涨，消息已吸收 → 追涨性价比低
- **未充分**：利好明确但板块反应平淡 → 挖掘机会的核心信号
- **过度**：涨幅远超合理影响 → 情绪炒作，回调风险

### 宏观三维坐标系

① 经济动能：改善/平稳/恶化
② 流动性环境：宽松/中性/收紧
③ 风险偏好：高/中/低

### 主要传导路径

| 事件类型 | 传导路径 | 影响板块 |
|---------|---------|---------| 
| 美联储表态 | 美元指数→外资偏好→北向资金 | 科技成长、外资重仓消费 |
| 国内货币政策 | MLF/LPR→融资成本→地产金融 | 银行、房地产 |
| 产业政策 | 受益板块→预期差 | 政策点名行业 |
| 大宗商品 | 油价→化工/能源/通胀 | 石油、化工、航空 |
| 地缘政治 | 风险偏好↓→避险 | 黄金、军工、银行 |

## Section 2：市场情绪温度计

**核心升级**：8维评分 + 散户/机构拆解 + 5日趋势

### 综合情绪评分（0-100）

| 指标 | 权重 | 评分标准 |
|------|------|---------|
| 全市场成交额 | 20% | 对比近20日均值，P70以上高分 |
| 涨停/跌停比 | 15% | 比值>5:高分, <0.5:低分 |
| 涨停封板率 | 15% | >85%强市, <70%弱市 |
| 两融余额变化 | 15% | 增加+分，减少-分 |
| 北向资金净额 | 15% | 流入+分，流出-分 |
| 全市场换手率 | 10% | 对比历史百分位 |
| 集合竞价偏离 | 10% | 隔夜集合竞价vs前收偏离 |

### 散户 vs 机构情绪分离

| 维度 | 散户情绪代理指标 | 机构情绪代理指标 |
|------|----------------|----------------|
| 数据 | 小单净流入、融资余额增速 | 超大单净流入、北向资金 |
| 关系判断 | 同向偏多=共识 / 散户热机构冷=情绪炒作 / 散户冷机构热=聪明钱建仓 |

### 情绪档位

| 得分 | 档位 | 操作含义 |
|-----|------|---------|
| 80-100 | 过热 | 注意顶部风险 |
| 60-79 | 乐观 | 趋势跟随 |
| 40-59 | 中性 | 结构性机会 |
| 20-39 | 悲观 | 等待反转信号 |
| 0-19 | 极度悲观 | 逆向参考 |

## Section 3：盘中时序解剖

> v2.0新增，基于量化研究的时段效应分析

### 6时段分解

| 时段 | 时间 | 核心含义 |
|------|------|---------|
| 集合竞价 | 9:15-9:25 | 隔夜定价偏好 |
| 早盘冲击 | 9:30-10:00 | 信息灵敏资金 |
| 上午主体 | 10:00-11:30 | 存量博弈 |
| 午后开盘 | 13:00-13:30 | 机构调仓窗口 |
| 午后主体 | 13:30-14:30 | 趋势确认 |
| 尾盘冲刺 | 14:30-15:00 | 主力意图 |

### 日内结构类型

- **A. 日内动量型** → 方向确定，次日延续概率高
- **B. 早盘反转型** → 情绪宣泄后理性回归
- **C. 尾盘异动型** → 主力表态，次日开盘最重要参考
- **D. 中午反转型** → 午休出现新信息或机构重评估

## Section 4：板块结构性分析

### 五维深度解剖（每个主线板块）

1. **驱动力诊断**：政策/业绩/主题/轮动/事件/估值修复
2. **轮动生命周期**：启动期→加速期→高潮期→分化期→衰减期
3. **资金主体识别**：游资/机构/量化（行为特征鉴别）
4. **产业链穿透**：上中下游联动分析 + 传导预测
5. **持续性评级**：★★★★★ 到 ★☆☆☆☆

### 涨停板生态

- 涨停总数 / 首板 / 连板 / 地天板
- 封板率 / 午前封板占比 / 尾盘炸板数
- 主题集中度 → 主线清晰度判断
- 龙头股：连板数、封板时间、封单量

## Section 5：资金路线图

### 全市场资金流量表

按超大单/大单/中单/小单拆解，判断：
- 超大单入+小单出 → 机构建仓散户减仓
- 超大单出+小单入 → 机构减仓散户接盘（危险）

### 资金迁移路线

流出板块 → 流入板块，判断迁移性质（防御→进攻/进攻→防御/行业内轮动/整体出场）

### 龙虎榜席位解读

识别机构专用席 vs 知名游资 vs 量化席位

## Section 6：技术形态诊断

### 均线与K线

- 指数技术状态矩阵（MA5/20/60/250 + 位置关系 + 评分）
- K线形态**概率性判断**（十字星: 转折55-65%, 锤头线: 反弹65-70%）
- 关键位计算：均线支撑 > 密集成交区 > 整数关口 > 斐波那契

### 量价关系 + 日内/隔夜分解

- 日内涨幅 = 收盘/开盘 - 1（真金白银推动）
- 隔夜涨幅 = 开盘/前收 - 1（情绪预期定价）

## Section 7：估值锚点追踪

> v2.0新增，机构最看重的底层驱动力

### 全市场估值水位

- PE(TTM) + 历史百分位（沪深300/创业板/科创50/中证500）
- P30以下=历史低位, P30-60=中位, P60以上=高位

### 盈利预期修正（Earnings Revision）

- 近1周上修/下修行业及原因
- 全A盈利预期变化方向

### 股债性价比（ERP）

- ERP = 沪深300股息率 - 10年期国债收益率
- ERP偏高 → 股票配置价值增强

## Section 8：次日多情景预判

三情景矩阵（概率之和=100%），每个情景必须包含：
1. 预期指数区间
2. 概率依据（≥3条）
3. **开盘后30分钟可观测的验证信号**
4. 操作参考

### 明日观测清单

🔴 最优先 → 🟡 重要 → 🟢 参考（含时间和影响方向）

## 写作质量控制

### 禁用表达黑名单（出现即重写）

| 禁用 | 替代 |
|------|------|
| "市场情绪有所回暖" | 情绪分从62升至65（+3），连续5日回升 |
| "需关注风险" | 沪指跌破3350点（MA20）则触发风险信号 |
| "可能震荡上行" | 60%概率沪指在3350-3420区间震荡 |
| "建议关注XX板块" | 若通信板块明日放量突破前高，确认第3日追踪 |
