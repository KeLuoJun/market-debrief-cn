## 第一层：宏观背景速递

**核心原则**：不是列新闻，是建立"事件→传导路径→今日盘面印证"的因果链条。

### 宏观事件筛选标准
- **入选**：当日对市场定价产生实际影响的事件
- **排除**：常规例行数据（无偏离预期）、重复报道的旧事件、纯噪音新闻

### 主要传导路径模板

| 事件类型 | 传导路径 | 影响板块 |
|---------|---------|---------|
| 美联储表态 | 美元指数变化 → 外资风险偏好 → 北向资金 → 成长/价值切换 | 科技成长、外资重仓消费 |
| 国内货币政策 | MLF/LPR调整 → 融资成本变化 → 地产/金融 | 银行、房地产、高息债替代 |
| 产业政策文件 | 受益板块明确 → 资金是否已开始定价（vs 预期差） | 政策点名行业 |
| 大宗商品价格 | 原油上涨 → 化工/能源成本/通胀 | 石油、化工、航空（反向） |
| 地缘政治 | 风险偏好下降 → 避险资产 → A股防御性配置 | 黄金、军工、银行 |
| 国内经济数据 | PMI超预期 → 顺周期 → 工业、材料 | 周期板块 |

### 分析写法示例
```
【事件】美联储3月会议纪要显示降息预期降温
传导路径：降息预期降温 → 美债收益率上升 → 美元走强 →
外资风险偏好下降 → 北向资金谨慎
盘面印证：今日北向净流出XX亿，与外资减持（指数基金重仓股）反应一致；
白酒/消费板块（北向重仓）今日偏弱。
```

## 第二层：市场情绪温度计

情绪评分由 `generate_report.py` 自动计算（0-100分），Claude主要负责**解读**：

### 情绪档位解读

| 得分 | 档位 | 市场含义 | 操作含义 |
|-----|-----|---------|---------|
| 80-100 | 过热 | 成交量放大、涨停潮、融资激增 | 注意顶部风险，警惕"量价背离" |
| 60-79 | 乐观 | 赚钱效应良好，主动买入积极 | 趋势跟随，但注意过热信号 |
| 40-59 | 中性 | 存量博弈，方向不明 | 结构性机会，选股大于选时 |
| 20-39 | 悲观 | 缩量、跌停多、融资减少 | 筑底阶段，等待信号反转 |
| 0-19 | 极度悲观 | 恐慌性抛售，跌停板大量出现 | 历史极端底部区域，逆向参考 |

### 关键异常情况识别
- **成交额突增但指数小涨**：可能是主力换手、大资金出逃迹象
- **涨停数多但成交额不高**：题材炒作为主，缺乏实质资金
- **北向大幅流入但指数平盘**：获利筹码抛压与外资买入对冲
- **两融余额明显增加**：杠杆资金入场，市场情绪偏积极但风险加大

### 短期情绪趋势（对比近20日）
- 若今日评分 > 近20日均值 +10分：情绪加速升温，关注过热风险
- 若今日评分 < 近20日均值 -10分：情绪快速降温，关注超跌反弹机会

## 第三层：板块结构性分析

### 驱动逻辑判断框架

**政策催化型**（持续性最强）
- 特征：政策文件明确点名、受益方向清晰、资金主要来自机构
- 判断：早盘开盘即涨、全天持续强势、换手率相对低
- 次日策略：龙头股具有较好持续性

**业绩驱动型**（持续性较强）
- 特征：财报季、业绩预告或数据超预期
- 判断：个股差异大、龙头股更强、量价配合良好
- 次日策略：业绩实质性公司可持续，但注意"高开低走"兑现

**主题炒作型**（持续性最弱）
- 特征：概念新颖、游资主导、换手率极高
- 判断：下午以来加速、涨停板数量多、板块内个股分化大
- 次日策略：首板效应强，但第二天开始分化加速

### 轮动阶段识别

| 阶段 | 信号 | 板块状态 |
|-----|-----|---------|
| 龙头启动期 | 细分板块最强个股开始异动 | 少数个股涨停，板块整体平 |
| 主升浪 | 龙头带领板块集体上涨 | 板块内普涨，量能放大 |
| 补涨接力 | 龙头调整，跟风股补涨 | 龙头高位震荡，小弟补涨 |
| 尾声分化 | 龙头跌停或大阴线，差股下跌 | 板块内强弱分化加剧 |

### 资金来源识别

| 资金类型 | 行为特征 | 涨停板特征 |
|---------|---------|----------|
| 游资 | 集中在低价小盘股、快进快出 | 封板力度大，但次日开板率高 |
| 机构 | 均线之上建仓、成交量温和放大 | 缓慢拉升，很少一字板 |
| 量化 | T+0高频、盘中波动大、换手率极高 | 多次反复封/开板 |
| 北向外资 | 白马蓝筹、持仓稳定 | 少有涨停，以净流入数据为准 |

## 第四层：资金路线图

### 资金归因分析框架

**北向资金**
- 净流入 > +50亿：外资积极看多，配合指数上涨则强势确认
- 净流入 10-50亿：温和流入，市场偏多
- -10亿至+10亿：中性，无明显偏向
- 净流出 > -50亿：外资减仓，注意板块影响（主要是消费/金融）

**两融数据**
- 融资余额增加：杠杆资金进场，情绪偏积极
- 融资余额连续3日增加：市场趋势性做多信号
- 融资余额单日变化 > 50亿：明显的资金动向，值得关注

**龙虎榜数据分析**
- 机构席位净买入大盘股：中线布局信号，含金量高
- 游资席位集中出现在某板块：该板块可能处于主升浪
- 机构席位净卖出但股价不跌：可能有其他资金对倒或接盘

### 资金迁移路线描述格式
```
今日资金路线：
资金流出：[板块A]（减仓原因）→ [板块B]（减仓原因）
资金流入：[板块C]（建仓逻辑）→ [板块D]（建仓逻辑）
主线资金：集中在[主线方向]，资金来源以[机构/游资/北向]为主
```

## 第五层：技术形态诊断

### 均线分析

指数轨迹判断（数据脚本自动计算MA5/10/20/60）：

| 位置关系 | 市场含义 |
|---------|---------|
| 价格 > MA20 > MA60 | 多头排列，趋势健康 |
| 价格 < MA20 < MA60 | 空头排列，趋势偏弱 |
| 价格在MA20下方但MA60上方 | 短期调整，中期趋势未破 |
| 价格下穿MA60 | 重要支撑破位，需警惕 |

### K线形态解读

| 形态 | 条件 | 含义 |
|-----|-----|-----|
| 十字星 | 实体/全幅 < 15% | 多空均衡，方向不明，需结合量能 |
| 大阳线 | 涨幅 > 3% | 多头主导，上行动能强 |
| 大阴线 | 跌幅 > 3% | 空头主导，调整压力大 |
| 锤头线 | 下影线 > 实体2倍 且收阳 | 下跌中的潜在反转信号 |
| 射击之星 | 上影线 > 实体2倍 且收阴 | 上涨中的潜在见顶信号 |

### 量价关系判断

| 量价组合 | 研判 |
|---------|-----|
| 放量涨 (量比>1.5) | 人气高涨，趋势延续概率大 |
| 缩量涨 (量比<0.7) | 多头惜售，但动能不足 |
| 放量跌 (量比>1.5) | 抛压重，调整可能延续 |
| 缩量跌 (量比<0.7) | 惜售为主，调整温和，底部信号 |

### 支撑压力位确定

优先级排序：
1. **均线支撑**：MA20 / MA60（权重最高）
2. **前期密集成交区**：价格在某区间内成交量集中
3. **整数关口**：如3000、3500等心理位
4. **斐波那契回撤位**：前一段行情的38.2%/50%/61.8%回撤

## 第六层：次日预判

### 预判规范

**禁止表述**（模糊，无意义）：
- "可能震荡上行"
- "维持偏多格局"
- "后市有望走高"

**必须格式**：
```
核心情景（60%概率）：
沪指在[具体区间]震荡，触发依据：[1-3个量化依据]
关注品种：[具体板块/方向]

乐观情景（25%概率）：
突破[具体价位]，触发条件需要：[具体条件，如北向流入超X亿 + 成交额放大至Xwx亿]

悲观情景（15%概率）：
跌破[具体支撑位]，触发因素：[具体风险事件或技术信号]
```

### 隔夜因素检查清单（次日预判前必看）
- [ ] 美股三大指数收盘表现
- [ ] 美元指数和人民币汇率
- [ ] 国际油价/金价变动
- [ ] 是否有重要经济数据发布（A股开盘前）
- [ ] 是否有政策文件或重要会议
- [ ] 前一日涨停板次日开板溢价情况（情绪延续性参考）

## 第七层：可回测的历史对比

### 相似时期识别维度

将以下当前数据与历史同期对比：
- 市场情绪评分（参考本报告第二层评分区间）
- 成交额范围（e.g. 1-1.5万亿）
- 涨停/跌停比（e.g. 涨停>跌停5倍以上）
- 北向资金方向（持续净流入>5日）
- 主板/创业板相对强弱
- K线形态（底部锤头/顶部射击之星等）

### 历史相似标注格式
```
当前市场状态类似于[历史时期，如2024年10月初/2023年2月]：
相似点：
- 情绪评分同处[X-Y分]区间
- 成交额连续[N]日在[X]亿上下
- [其他相似指标]
後续表现（历史参考）：
- 5日后上涨概率：X%，平均涨幅：Y%
- 10日后上涨概率：X%，平均涨幅：Y%
```

### 上期预判验证格式
```
上期预判（[日期]）：
核心情景（60%）：[当时预判内容]
实际结果：[实际价格和方向]
评估：[准确/方向准确但区间偏差/方向错误]
修正：[若预判失误，分析原因和改进]
```

## 情绪评分计算规则

| 指标 | 规则 | 最大贡献 |
|-----|-----|---------|
| 成交额 | >1.5万亿:+15; >1.2万亿:+10; >1万亿:+5; <8000亿:-10; <5000亿:-15 | ±15 |
| 涨跌停比 | 比值>5:+12; 比值>2:+8; 比值<0.5:-8; 比值<0.2:-12 | ±12 |
| 北向资金 | 净流入>50亿:+10; 流入>0:+5; 流出>50亿:-10; 流出>0:-5 | ±10 |
| 两融变化 | 增加>50亿:+10; 增加>0:+5; 减少>50亿:-10; 减少>0:-5 | ±10 |
| 换手率 | >3%:+7; >2%:+3; <1%:-5 | ±7 |

基础分50 + 各项得分，最终限制在0-100分。

## 常见分析陷阱

1. **事件新闻化**：把新闻直接陈列而不分析传导路径 → 必须写因果链
2. **情绪不量化**：用"市场偏暖"代替具体评分 → 必须给出0-100分
3. **板块不归因**：只写"某板块涨了X%"而不解释为什么 → 必须说明驱动逻辑
4. **预判无区间**：写"明日可能继续上涨"而不给价格区间 → 必须给具体点位
5. **无概率分布**：只给单一情景 → 必须给60%/25%/15%三情景
6. **不验证上期预判**：每次都当第一次分析 → 7层必须包含上期验证
7. **技术分析空洞**：只写"位于MA20上方"而不说明意义 → 必须解释多空含义
