## Module 1: 宏观定价扫描

**核心问题**：今日行情背后的驱动力是什么？市场是否已充分定价？

**深度要求**：
- 每个有效事件必须追溯**二阶影响**：事件→直接板块→外溢至哪些相关板块
- 定价完成度判断要引用**具体数据**（板块涨幅%、换手率、与30日均值比较倍数）
- 隔夜外盘必须给出**具体的传导判断**（哪类股票今日开盘可能受影响、正面/负面/中性）
- 不得仅罗列新闻标题——每条有效事件必须有盘面数据印证，两者缺一不可

### 1.1 有效事件过滤表

从 tavily 新闻中筛选对盘面有真实传导影响的事件，按以下四列呈现：

| 列 | 说明 |
|----|------|
| 事件 | 简述，≤30字 |
| 传导路径 | 事件→直接受影响板块→二阶外溢板块 |
| 盘面印证 | 具体板块涨幅%、龙头股表现、成交量倍数 |
| 定价状态 | 已充分 / 未充分 / 过度（附具体判断依据） |

**判断标准**：
- 已充分定价：板块涨幅与事件量级匹配，成交量放大但未异常
- 未充分定价：事件重大但对应板块涨幅低于预期，或尚未反应→潜在机会
- 过度定价：板块涨幅远超合理估算影响，换手率>5%→回调风险

**筛选标准**（至少满足其一才列入）：
- 对应板块当日涨幅 ≥ +2%（与事件正传导）
- 对应板块当日跌幅 ≥ -2%（与事件负传导）
- 成交额异常放大 ≥ 事件前3日均值的1.5倍

### 1.2 宏观三维坐标系

每日更新三个状态坐标，**每个坐标必须附1条数据依据**：

| 坐标 | 状态选项 | 判断依据 |
|------|---------|---------|
| 经济动能 | 改善 / 平稳 / 恶化 | 最近 PMI、工业增加值、社融数据方向 |
| 流动性环境 | 宽松 / 中性 / 收紧 | MLF/OMO操作、Shibor走势、两融余额变化（亿元） |
| 风险偏好 | 高（>70） / 中（40-70） / 低（<40） | 基于情绪综合分 |

**坐标变化方向**：若今日与昨日不同，标注"↑改善"或"↓恶化"。

### 1.3 隔夜外盘摘要

覆盖：美股三大指数、原油（WTI）、黄金（COMEX）、铜（LME）。
每项输出：收盘价（或期货价）、当日涨跌幅%、**对A股次日传导判断**（具体受影响板块和方向）。

**传导判断模板**：
- 美股科技大涨 → A股半导体/AI板块高开概率↑，但需警惕回调风险
- 原油暴涨 → A股石油石化可能高开，但通胀预期压制消费板块
- 铜价上涨 → 工业金属、电力设备板块受益

## Module 2: 市场情绪温度计

**核心问题**：今日市场参与者的整体情绪处于什么水位？散户和机构是否分歧？

**深度要求**：
- 综合情绪分必须**精确到个位数**，各分项得分也要全部列出
- 散户/机构分歧类型必须给出**行为推断**（谁在减仓、谁在加仓、筹码交换还是单向流动）
- 5日趋势要分析**趋势斜率**（加速上行/高位走平/见顶回落/下行趋稳）
- 必须有**历史对比**：当前情绪分处于近60日的什么水位（百分位）

### 2.1 综合情绪评分（0-100）

| 指标 | 权重 | 属性 | 计算方法 |
|------|------|------|---------|
| 全市场成交额/20日均值 | 20% | 综合 | ratio=1.0→50分；ratio=1.5→80分；ratio=0.5→20分；线性插值 |
| 涨停/跌停比 | 15% | 散户 | ratio≥5→90分；ratio=1→50分；ratio≤0.2→10分 |
| 涨停封板率 | 15% | 散户 | 封板率≥80%→90分；50%→50分；≤20%→10分 |
| 两融余额变化 | 15% | 散户 | 日增≥50亿→80分；增0→50分；日减≥50亿→20分 |
| 北向资金净额 | 15% | 机构 | 净买入≥50亿→90分；0→50分；净卖出≥50亿→10分 |
| 全市场换手率 | 10% | 综合 | 与20日均值对比，同成交额算法 |
| 超大单净流入方向 | 10% | 机构 | 正→70分；0附近→50分；负→30分 |

**附加输出**：
- 60日情绪分百分位（当前分高于近60日多少%交易日）
- 趋势斜率描述：连续3日方向 → "加速上行/高位走平/拐头下行/筑底反弹"

### 2.2 散户/机构情绪拆解

- **散户情绪代理分** = (涨停跌停比×0.33 + 封板率×0.33 + 两融变化×0.34) 的加权均值
- **机构情绪代理分** = (北向资金×0.5 + 超大单方向×0.5) 的加权均值

**分歧判断及行为推断**：

| 散户情绪 | 机构情绪 | 判断 | 行为推断 |
|---------|---------|------|---------|
| >60 | >60 | 共识多头 | 各路资金同向流入，市场强势 |
| <40 | <40 | 共识空头 | 整体撤离，防守为主 |
| >60 | <40 | 散户热机构冷 | ⚠️ 机构在出货，散户在接盘——警惕追高 |
| <40 | >60 | 散户冷机构热 | 👀 机构低吸布局，散户恐慌出逃——关注低吸 |

### 2.3 5日情绪趋势

输出近5个交易日的情绪分序列，计算并标注：
- 5日均值、20日均值
- 当综合分 > 60日均值 + 1σ 时，标注「追涨风险区间」（加入markArea）
- 60日分位：「当前60分处于近60日的P45，情绪中等」

## Module 3: 板块结构性分析

**核心问题**：今日谁在涨、为什么涨、处于轮动的哪个阶段？

**深度要求**：
- TOP2板块解剖必须给出**龙头股名称**（涨停数、涨幅领先的2-3只）
- 轮动阶段判断要给出**持续天数**（板块已连涨/连跌X天）
- 赚钱效应判断要结合**连板高度**（最高几连板、有无涨停炸板的龙头）
- 成长/价值风格要给出**量化对比**（代表行业均涨幅差）

### 3.1 行业强弱全景

按申万一级行业涨跌幅分为四档，**必须列出各档代表行业及当日涨跌幅**：

| 档位 | 涨跌幅区间 | 代表行业（含涨跌幅） |
|------|-----------|-----------------|
| 强势 | ≥ +2% | 如：电力设备+3.2%、化工+2.6% |
| 次强 | +0.5% ~ +2% | ... |
| 中性 | -0.5% ~ +0.5% | ... |
| 弱势 | ≤ -0.5% | ... |

**成长/价值风格对比**（必须有数字）：
- 成长均涨幅 vs 价值均涨幅
- 差值 > +1% → 成长占优；< -1% → 价值占优；[-1%,+1%] → 均衡

### 3.2 TOP 2 板块深度解剖

选取标准：当日涨幅和净流入均排名前5的交集中，取涨幅最高的2个。

**五维度解剖**（新增龙头股维度）：

| 维度 | 选项/内容 |
|------|---------|
| 驱动力来源 | 政策/业绩/主题/轮动承接/估值修复/事件催化（说明具体原因）|
| 轮动周期 | 启动/加速/高潮/分化/衰减（标注已持续N天） |
| 资金主体 | 游资（龙虎榜特征）/ 机构（大单为主）/ 量化 / 混合 |
| 龙头股 | 今日涨停/涨幅领先的2-3只，注明连板高度 |
| 持续性评级 | ★★★（高/中/低，附2条理由） |

**轮动周期判断依据**：
- 启动：板块连续涨≤2日，成交量刚开始放大
- 加速：板块连涨3-5日，量能持续递增，龙头股开始涨停
- 高潮：单日涨幅最大，涨停数最多，成交量创阶段新高
- 分化：板块仍涨但个股分化加剧，涨停数减少，高位股开始回落
- 衰减：板块整体走弱，龙头股补跌，资金明显撤离

### 3.3 涨停板生态快照

输出指标（**均要有具体数字**）：
- 涨停总数、首板数、2连板数、3连板及以上数、最高连板数（N连板龙头名）
- 封板率 = 涨停数 / (涨停数 + 炸板数) × 100%
- 炸板数
- 涨停股主题集中度 = 最大主题涨停数 / 涨停总数（如「能源赛道占涨停35%」）

**赚钱效应判断**：

| 封板率 | 涨停数 | 连板高度 | 赚钱效应 |
|-------|-------|---------|---------|
| ≥70% | ≥50 | ≥5板 | 强（积极入场信号）|
| 50-70% | 30-50 | 3-4板 | 中（精选题材）|
| <50% | <30 | ≤2板 | 弱（观望为主）|

## Module 4: 资金路线图

**核心问题**：钱从哪来、去了哪里、主力意图是什么？

### 4.1 全市场资金结构

| 计算项 | 公式 |
|-------|------|
| 机构行为方向 | sign(超大单净流入 + 大单净流入) |
| 散户行为方向 | sign(小单净流入) |
| 中单缓冲 | 中单净流入方向（趋势跟随判断） |

**行为组合分类**：

| 机构方向 | 散户方向 | 类型 |
|---------|---------|------|
| 流入 | 流入 | 共识多头 |
| 流出 | 流出 | 共识空头 |
| 流入 | 流出 | 机构建仓、散户减仓 |
| 流出 | 流入 | 机构减仓、散户接盘 ⚠️ |

### 4.2 资金迁移路线

1. 取净流出额 TOP3 行业（来源端）
2. 取净流入额 TOP3 行业（目标端）
3. 判断迁移性质：

| 来源端特征 | 目标端特征 | 迁移性质 |
|-----------|-----------|---------|
| 进攻性板块 | 防御性板块 | 进攻→防御（Risk-off） |
| 防御性板块 | 进攻性板块 | 防御→进攻（Risk-on） |
| 同类板块 | 同类板块 | 行业内轮动 |
| 各板块均流出 | 无明显流入 | 整体出场 |

### 4.3 龙虎榜席位解读

席位分类依据：
- **知名游资**：量化名单以外的、高净额打板席位（如宁波桑田路、拉萨团结路等）
- **机构专用**：席位名称含「机构专用」
- **量化席位**：高频进出、金额均匀分批的营业部

## Module 5: 技术形态与估值

**核心问题**：市场在哪里、贵不贵、下一步关键位在哪？

### 5.1 技术状态矩阵

对沪深300、创业板指、科创50各计算：

| 指标 | 计算 |
|------|------|
| MA5偏离度 | (收盘 - MA5) / MA5 × 100% |
| MA20偏离度 | (收盘 - MA20) / MA20 × 100% |
| MA60偏离度 | (收盘 - MA60) / MA60 × 100% |
| MA250偏离度 | (收盘 - MA250) / MA250 × 100% |
| 均线排列 | 多头(MA5>MA20>MA60) / 空头(反序) / 粘合(偏离<1%) |
| 量比 | 今日成交量 / 5日平均成交量 |
| K线形态 | 识别名称（长上影、十字星、大阳线等） |

### 5.2 K线形态识别与概率解读

| 形态 | 识别条件 | 历史统计参考 |
|------|---------|------------|
| 大阳线 | 实体涨幅 > 2%，上下影线短 | 后3日继续上涨概率 ~55% |
| 大阴线 | 实体跌幅 > 2%，上下影线短 | 后3日继续下跌概率 ~50% |
| 长上影 | 上影线 > 实体2倍 | 3日内回调概率 ~65% |
| 长下影 | 下影线 > 实体2倍 | 3日内反弹概率 ~60% |
| 十字星 | 实体 < 0.3%，有上下影 | 变盘信号，方向看后续量能 |
| 锤子线 | 下影线长，小实体在上方 | 底部反转信号概率 ~58% |

> 概率数据为经验估算，需结合所处位置（高位/低位）和量能综合判断。

### 5.3 关键支撑/压力位

每个关键位标注计算依据：

| 类型 | 来源 |
|------|------|
| MA支撑/压力 | 5日/20日/60日/250日均线位置 |
| 斐波那契 | 近期高低点的38.2%/50%/61.8%回撤位 |
| 成交密集区 | 近30日成交量加权平均价（VWAP近似） |
| 整数关口 | 如3000、3500、4000等心理关口 |

### 5.4 量价关系判断

| 今日涨跌 | 量比 | 判断 |
|---------|------|------|
| 涨 | >1.2 | 放量上涨，多头强势 |
| 涨 | <0.8 | 缩量上涨，上攻动力不足 |
| 跌 | >1.2 | 放量下跌，空头强势 |
| 跌 | <0.8 | 缩量下跌，恐慌有限 |

**日内/隔夜涨幅区分**：
- 隔夜涨幅 = 开盘价/前收盘价 - 1（反映外盘/消息面影响）
- 日内涨幅 = 收盘价/开盘价 - 1（反映盘中资金行为）

### 5.5 估值水位表

| 指数 | 输出项 |
|------|-------|
| 沪深300 | PE(TTM)、历史百分位 |
| 中证500 | PE(TTM)、历史百分位 |
| 中证1000 | PE(TTM)、历史百分位 |
| 上证50 | PE(TTM)、历史百分位 |

百分位判断档位：
- P30以下 → 历史低位（具备安全边际）
- P30-P60 → 历史中位（估值中性）
- P60以上 → 历史高位（需警惕估值风险）

### 5.6 股债性价比（ERP）

```
ERP = 沪深300股息率 - 10年期国债收益率
```

沪深300股息率近似 = 1 / PE(TTM)（当个别年份分红率变化大时此法有偏差，但可用于趋势判断）

| ERP | 判断 |
|-----|------|
| > 历史均值 + 1σ | 股票极具配置价值 |
| > 历史均值 | 股票性价比偏高 |
| ≈ 历史均值 | 中性 |
| < 历史均值 | 债券性价比更优 |

## Module 6: 次日预判与历史镜像

**核心问题**：明天最可能发生什么、历史上类似时期之后怎样了？

**深度要求**：
- 三情景概率之和 = 100%，**每个情景必须含指数预期区间**（如「上证4080-4130」）
- 触发条件必须是**开盘30分钟内可观测**的具体信号（成交量、价格、北向方向）
- 历史镜像必须使用**真实历史区间**（如「2024年2月7日-14日」），不得模糊为「某月某时」
- 若有上期报告，必须做**准确度回顾**（实际发生了哪个情景，预测是否准确）

### 6.1 三情景预判

| 情景 | 概率范围 | 必含要素 |
|------|---------|---------|
| 乐观情景 | 20-30% | 指数区间、成交量区间、≥2条触发条件 |
| 核心情景 | 55-65% | 指数区间（上下20点）、成交量区间、主要逻辑 |
| 悲观情景 | 10-20% | 指数区间、≥2条高确定性触发信号 |

**概率依据要求**（每个情景注明依据）：
- 技术面依据：均线位置（具体点位）、量比水平、K线形态概率
- 情绪依据：情绪分水位、散户/机构分歧方向
- 事件依据：次日已知事件（数据发布时间、政策窗口等）

**触发条件必须使用量化信号**，示例：
- ✅「创业板开盘30分钟成交额 < 200亿且跌幅 > 1%」
- ✅「沪深300早盘净主力流入 > 50亿以上」
- ❌「外盘表现良好」（太模糊）

**三情景概率之和必须 = 100%**

### 6.2 次日重点观测清单

按优先级列出，每项必须包含「观测对象 + 具体阈值 + 应对措施」：

| 优先级 | 观测项 | 阈值/信号 | 应对 |
|--------|--------|---------|------|
| 高 | 创业板开盘强弱 | 跌破3250 vs 站稳3300 | 具体操作 |
| 高 | 成交额速度 | 10点前万亿/900亿速率 | 具体应对 |
| ... | ... | ... | ... |

### 6.3 历史镜像

匹配维度：
- 估值百分位（±10%范围）
- 情绪分（±15分范围）
- 趋势状态（均线排列一致）

输出要求（**必须精确**）：
- 历史参照期：具体日期区间（如「2024年2月5-15日」）
- 核心相似点：2-3条，每条附数据对比
- 关键差异：1-2条（影响类比可靠性的因素）
- 该历史参照期后 5日/10日/20日 平均表现（用当时实际数据）

### 6.4 上期预判回顾

若存在上期报告，**必须做这一步**（不可省略）：

| 项目 | 内容 |
|------|------|
| 上期核心情景 | [描述] |
| 实际发生情景 | 核心/乐观/悲观 |
| 关键位准确度 | 支撑/压力位命中率 |
| 准确度评级 | 高/中/低 |
| 误判原因 | ≤50字 |
